## Stage Definitions

| Stage | Description |
|-------|-------------|
| `Planning` | Approved idea, not yet scripted |
| `Scripting` | Outline or script in progress |
| `Recording` | Script done, recording in progress |
| `Editing` | Recorded, edit in progress |
| `Review` | Edit done, final review |
| `Scheduled` | Uploaded, scheduled for publish |
| `Published` | Live on YouTube |
| `Archived` | Old video, no longer tracking actively |
