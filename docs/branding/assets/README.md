# Brand Assets

This directory contains brand assets for the project.

## Organization

```
assets/
├── logo/
│   └── selected/
│       └── v1/              # Current selected logo pack
│           ├── source/      # Original source files
│           ├── png/         # Exported PNG formats
│           │   ├── full/    # Full logo (icon + wordmark)
│           │   ├── icon/    # Icon only (square/tight variants)
│           │   └── wordmark/ # Wordmark only (horizontal text)
│           └── docs/        # Usage documentation
```

## Canonical Files

The following files are the recommended choices for common use cases:

| Use Case | File Path |
|----------|-----------|
| **Avatar / Profile** | `logo/selected/v1/png/icon/logo-avatar-square-800.png` |
| **Favicon** | `logo/selected/v1/png/icon/favicon.ico` |
| **Full Logo (Medium)** | `logo/selected/v1/png/full/logo-full-512.png` |
| **Full Logo (Large)** | `logo/selected/v1/png/full/logo-full-1024.png` |
| **Wordmark (Web)** | `logo/selected/v1/png/wordmark/logo-wordmark-1200w.png` |
