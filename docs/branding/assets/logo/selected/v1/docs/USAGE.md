# Selected Logo Asset Pack (v1)

This pack contains PNG derivatives generated from the chosen logo image.

## Folders
- `source/`
  - `original.jpeg` — the exact input
  - `original-transparent.png` — background removed
- `png/full/`
  - `logo-full-*.png` — full mark (icon + text)
- `png/icon/`
  - `logo-icon-tight-*.png` — icon only, tight crop
  - `logo-avatar-square-*.png` — icon only, padded square for avatars
  - `favicon.ico` — multi-size ICO built from the square icon
- `png/wordmark/`
  - `logo-wordmark-*.png` — text portion only (good for headers/banners)

## Recommended usage
- YouTube avatar: `png/icon/logo-avatar-square-800.png`
- Website favicon: `png/icon/favicon.ico`
- Docs/headers: `png/full/logo-full-512.png` or `png/wordmark/logo-wordmark-1200w.png`

## Note
This is a raster-derived pack. If you need true vector (SVG), you’ll need to recreate/trace the icon in a vector editor.
