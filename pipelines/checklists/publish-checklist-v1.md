## Pre-Upload

### Video File
- [ ] Final export rendered
- [ ] Watched full video for quality check
- [ ] Audio levels consistent throughout
- [ ] No unintended silence or dead air
- [ ] Graphics and overlays display correctly
- [ ] End screen safe zone clear (last 20 seconds)

### Thumbnail
- [ ] Thumbnail created (1280x720)
- [ ] Text readable at small size
- [ ] Matches brand guidelines
- [ ] Saved to `assets/thumbnails/`

---

## YouTube Upload

### Basic Info
- [ ] Video uploaded (unlisted or scheduled)
- [ ] Title finalized
  - [ ] Front-loaded with keywords
  - [ ] Under 100 characters
  - [ ] Accurate to content (not clickbait)
- [ ] Description complete
  - [ ] Summary in first 150 characters
  - [ ] Timestamps/chapters added
  - [ ] Resources linked
  - [ ] Disclosures included (if applicable)
  - [ ] Hashtags added (max 15)
- [ ] Tags added (if using)

### Thumbnail
- [ ] Custom thumbnail uploaded
- [ ] Verified thumbnail displays correctly in preview

### Playlist
- [ ] Added to relevant playlist(s)

---

## Video Elements

### End Screen
- [ ] End screen added
- [ ] Subscribe element included
- [ ] Video/playlist element included
- [ ] Elements don't overlap important content

### Cards
- [ ] Cards added at relevant moments
- [ ] Links to related videos or playlists
- [ ] Card text is clear and relevant

### Subtitles/Captions
- [ ] Auto-captions enabled
- [ ] Reviewed and edited auto-captions (recommended)
  OR
- [ ] Uploaded custom captions file

---

## Final Verification

### Quality Check
- [ ] Preview video one more time
- [ ] Thumbnail displays correctly in all sizes
- [ ] Title and description accurate
- [ ] All links work
- [ ] No spelling errors

### Compliance
- [ ] Content complies with YouTube policies
- [ ] All music is licensed
- [ ] All claims are accurate and sourced
- [ ] Disclosures are complete

### Tracking
- [ ] Episode tracker updated (`tracking/episodes/episodes.md`)
- [ ] Stage set to "Published" (or "Scheduled")
- [ ] Planned publish date recorded

---

## Post-Publish (After Video Goes Live)

### Immediate (Within 1 Hour)
- [ ] Verify video is live and public
- [ ] Post pinned comment
- [ ] Share on social media
- [ ] Watch first few minutes for playback issues

### 24 Hours
- [ ] Check comments, respond to early engagement
- [ ] Review initial analytics (views, CTR)
- [ ] Note any issues or feedback

### 7 Days
- [ ] Review full analytics
- [ ] Update episode tracker with metrics
- [ ] Add retro notes
