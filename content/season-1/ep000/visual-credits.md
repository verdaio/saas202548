## Rights-Safe Visual Sourcing SOP

### Default Visual Stack (in order of preference)

1. **House-made visuals:** diagrams, maps, icons, overlays (preferred)
2. **Public domain / government visuals:** with attribution
3. **Licensed stock:** only if license is clear and stored
4. **Company logos/screenshots:** limited, factual, used as identifiers, with attribution

## Maps Policy Checklist

- [ ] Prefer schematic maps (pins/paths) over map tiles
- [ ] If using map imagery/tiles, license allows it and attribution is followed
- [ ] Map overlays are minimal (highlight flows, not detailed cartography)

## Logos/Screenshots Policy Checklist

- [ ] Used only when necessary to identify entities or show documented interfaces
- [ ] No endorsement implied
- [ ] Prolonged logo display avoided (keep it contextual)
